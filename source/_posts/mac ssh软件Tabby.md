# 软件介绍

Tabby 是一名老外在 Github 开源的终端连接的工具(ssh工具)，至今已经累积 45K+ star。

Tabby 的功能特性大概有：

- 支持多平台，Windows、MacOS（Intel 芯片/M1 芯片）、Linux 都有对应的安装包的；
- 自带 SFTP 功能，能够与 Linux 系统传输文件；(rz sz命令)
- 炫酷的终端页面，简单易用，以及各种插件支持等
